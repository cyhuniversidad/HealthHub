Declaración de accesibilidad - HealthHub

Compromiso con la accesibilidad

En HealthHub, estamos comprometidos en garantizar la accesibilidad de nuestro sitio web para todas las personas, independientemente de sus capacidades. Nos esforzamos por cumplir con las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1 en su nivel A y con el Real Decreto 1112/2018.

Características de accesibilidad implementadas

  • Identificación del idioma: Todas las páginas de nuestro sitio están correctamente etiquetadas con el atributo de idioma para facilitar la navegación con lectores de pantalla.
  • Texto alternativo: Las imágenes cuentan con texto alternativo descriptivo para garantizar que el contenido visual sea accesible para personas con discapacidades visuales.
  • Navegación clara: Hemos implementado un mapa del sitio para facilitar la navegación por todo el contenido disponible en HealthHub.
  • Estructura semántica: Usamos etiquetas HTML5 semánticas para garantizar que la estructura del contenido sea comprensible para tecnologías asistivas.
  • Contraste adecuado: El diseño del sitio se optimiza para garantizar un contraste adecuado que mejore la legibilidad del contenido para personas con baja visión.
  • Acceso por teclado: Todos los elementos interactivos son accesibles mediante teclado, para usuarios con discapacidades motoras.

Mecanismos de retroalimentación

Si experimentas dificultades para acceder a cualquier parte de nuestro sitio web o deseas proporcionar comentarios sobre la accesibilidad, no dudes en contactarnos a través del Formulario de ayuda al cliente.

Niveles de conformidad

Este sitio web cumple en su mayoría con el nivel A de las Pautas de Accesibilidad para el Contenido Web (WCAG) 2.1, aunque aún estamos trabajando en mejorar ciertos elementos para garantizar una accesibilidad total.
